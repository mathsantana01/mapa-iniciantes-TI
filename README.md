# Resumos estruturados
- Tornar-se um engenheiro de software é uma jornada que exige dedicação e aprendizado contínuo.

- Dominar a lógica de programação é fundamental no início de quem quer entrar na área.

- Antes de escolher estudar várias linguagens, focar em uma linguagem de programação é essencial.

- Um Engenheiro de Software é o profissional responsável por projetar, desenvolver e manter sistemas, aplicativos e plataformas digitais

- Diferente de um programador focado apenas na escrita de código, um engenheiro de software cuida da arquitetura, escalabilidade, segurança e manutenibilidade 
ao longo de todo o ciclo de vida do software

- Construir projetos práticos será fundamental. Crie seu portfólio.

- Quer um diferencial competitivo? Desenvolva também habilidades emocionais (SoftSkills)
  

# Glossário

Lógica de Programação:
Capacidade de organizar o raciocínio para criar soluções por meio de código.

Engenharia de Software:
Área responsável por projetar, desenvolver, testar e manter sistemas de software de forma organizada e eficiente.

Desenvolvedor Full Stack:
Profissional capaz de desenvolver tanto o Front-end quanto o Back-end de uma aplicação.

Front-end:
Parte da aplicação com a qual o usuário interage diretamente.

Back-end:
Parte responsável pela lógica de negócio, processamento de dados e comunicação com o banco de dados.

Arquitetura de Software:
Forma como um sistema é estruturado e organizado para facilitar sua evolução e manutenção.

Escalabilidade:
Capacidade de um sistema suportar o aumento de usuários ou de demanda sem perder desempenho.

Soft Skills:
Habilidades comportamentais, como comunicação, trabalho em equipe, organização e resolução de problemas.
